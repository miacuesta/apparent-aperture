# Proyecto Astro
Mia Cuesta Barizom, 5C TIC

## Descripción
Este proyecto consiste en una aplicación web estática desarrollada con Astro que permite gestionar tareas de manera simple e interactiva.
El usuario puede agregar tareas, marcarlas como completadas o eliminarlas, además de interactuar con un menú de opciones que mejora la experiencia de uso.

## Herramientas
- Astro
- HTML
- CSS
- JavaScript

## Items
- Agregar tareas dinámicamente
- Marcar tareas como completadas con un click
- Eliminar tareas con doble click
- Borrar todas las tareas desde el menú
- Marcar todas las tareas como completadas
- Modo oscuro (dark mode)
- Menú lateral de opciones
- Animaciones al interactuar con la app

## Requisitos de la consigna cumplidos

-  1 página principal
-  Al menos 3 secciones (Agregar tarea, Lista de tareas, Sobre la app)
-  Diseño responsive básico
-  Uso de JavaScript (eventos y manipulación del DOM)

  ## Funcionalidades extra

- Componentes reutilizables (Card)
- Animaciones CSS y JS
- Menú interactivo
- Modo oscuro
- Mejora de UX/UI
- Deploy a Vercel

## Cómo ejecutar

1. instalar dependencias
npm install

2. correr proyecto
npm run dev

3. correr el local host dado en la url del navegador
